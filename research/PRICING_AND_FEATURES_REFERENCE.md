Credits consumed based on:
- **Text generation:** ~1 credit per 1,000 tokens (varies by model)

## CUSTOMER SEGMENT POSITIONING

### AbacusAI

**Best for:**
- Individuals replacing multiple AI subscriptions (ChatGPT + Claude + separate image gen)
- Teams wanting unified LLM access with minimal setup
- Businesses building internal AI workflows (via DeepAgent)
- Users who value LLM diversity and flexibility

**Weakest for:**
- Organizations with strict privacy/data residency requirements
- Users wanting agent identity or memory
- Enterprises needing self-hosted options

**Weakest for:**
- Users wanting agent identity or personality
- Organizations needing self-hosted privacy

## COMPETITIVE DYNAMICS SUMMARY

### Direct Competition: NONE (yet)

- **AbacusAI vs Perplexity:** Different value props. AbacusAI = unified access; Perplexity = search + reasoning. Complementary.
- **Either vs We Prosper AI:** Different positioning. They commoditize LLM access. We own agent identity + memory. Only competitive if they add these features.

### Indirect Competition: HIGH

- **ChatGPT Pro** ($20/mo) - Could eventually add memory + team features
- **Claude Pro** ($20/mo) - Could eventually add team features + memory
- **OpenAI's Operator** (free/built-in) - Could evolve toward persistent agents
- **Custom LLM wrappers** (Point solutions) - Building simple agents on top of APIs

### Market Convergence (12+ months)

Expect:
1. **AbacusAI** to add memory/identity features to DeepAgent
2. **Perplexity** to add persistent context layer to Agent API
3. **OpenAI/Anthropic** to add team/memory features to their platforms

**Our defensibility:** Move fast on identity + memory. Build such strong product that competitors can't easily copy the "agent as employee" paradigm.

## RECOMMENDED PRICING STRATEGY FOR WE PROSPER AI

### Phase 1 (Launch): Penetration pricing
- $299/mo for first 50 customers (introductory)
- After 50: Increase to $399/mo (standard pricing)
- Discount: 20% off annual (pay $3k for year, save $359.80)

### Phase 2 (Expansion): Value-based pricing
- Individual: $399/mo (1 agent)
- Team: $299/mo base + $99/agent (up to 5 agents)
- Enterprise: Custom (typically $5k-15k/mo)

### Pricing rationale:
- **Not per-user:** We're not LLM access. We're agent ROI.
- **Not per-token:** Customers don't care about token consumption. They care about conversions, meetings booked, costs saved.
- **Per-agent:** Each agent has value independent of team size. One agent in a 100-person org is still valuable.
- **Entry point is affordable:** $299/mo is <$20/day. Easy for VP Sales to trial without procurement.
- **Grows with success:** Customer adds 2nd agent because 1st is working. Natural upsell.
