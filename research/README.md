## Key Findings (Summary)

### Market Reality
- **TAM:** AI agent market: $52.6B by 2030 (46% CAGR from $7.84B in 2025)
- **Enterprise adoption:** 85% of enterprises deploying agents by 2025
- **Vertical agents:** Growing 35% CAGR (specialized > generic)
- **Sales agents specifically:** $47.5B market by 2034

### Competitive Positioning
| Platform | Strength | Weakness | Defending Against |
|----------|----------|----------|-------------------|
| **AbacusAI** | Multi-model consolidation + easy setup | No memory, no identity, cloud-only | Could add these in 9-12 months |
| **Perplexity** | Real-time search + agentic orchestration | No memory, no identity, cloud-only | Could add these in 3-6 months |
| **We Prosper AI** | Persistent memory + identity + self-hosted | Nascent product, no installed base | Move fast; build switching costs |

### Our Differentiation
1. **Persistent memory** (agents learn from interactions)
2. **Agent identity** (named agents with custom voices and personalities)
3. **Self-hosted option** (privacy-first deployment)
4. **Multi-agent orchestration** (family of agents working together)

These are the only defensible differentiators. Everything else (LLM access, search, integrations) is commoditizing.

### Recommended Strategy
- **Short game (6 months):** Launch sovereign sales agent on Perplexity backbone
- **Entry point:** $299-399/mo per agent (not per-user)
- **Go-to-market:** Direct to VP Sales / CROs (B2B SaaS focus)
- **Success metric:** 100 paying customers by month 12
- **Expected ARR:** $3.6M by EOY 2026

### Long Game (12-24 months)
- Build proprietary search layer (replace Perplexity)
- Build proprietary model orchestration (replace AbacusAI)
- Keep proprietary: Memory, identity, voice, multi-agent workflows
- Path to $100M+ via vertical expansion (support agents, ops agents, etc.)

## Critical Success Factors

1. **Speed:** MVP in 12 weeks, not 24. Move faster than Perplexity/AbacusAI can copy.
2. **Customer obsession:** Lock in first 10 customers with such valuable product that switching is emotional + costly.
3. **Build switching costs:** Persistent memory + custom voice = relationship. Hard to replace.
4. **Prove ROI:** Every customer should see measurable business outcome (conversions up, costs down, time saved).
5. **Defensible moat:** Identity + memory + self-hosted = hard to copy in 6-12 months.

## Questions & Clarifications

### Q: Why not start with enterprise sales (higher ACV)?
**A:** Sales cycle too long (6-12 months), CAC too high. Better to land-and-expand from SMB/founders who buy quickly and refer.

### Q: What if Perplexity adds memory to Agent API within 6 months?
**A:** Differentiation moves to voice + personality + multi-agent orchestration. We focus on being "AI employee" platform, not just "agent with memory."

### Q: Why target sales agents specifically?
**A:** Highest ROI use case (36% conversion lift documented). Sales teams have budgets. Success is measurable. Easy to sell.

### Q: Can we build this without Perplexity? Go fully proprietary?
**A:** Yes, but 2x longer timeline (18 weeks for MVP instead of 12). Recommend complement first, compete later.

### Q: What's the churn risk if we go cloud-only?
**A:** Moderate-to-high. Self-hosted option is differentiator. Include it in MVP.

### Q: How much runway do we need?
**A:** $750k to reach 50 paying customers and profitability. Can fundraise at month 4 if traction is strong ($3.6M ARR).
