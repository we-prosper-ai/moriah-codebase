## EXECUTIVE SUMMARY

Three distinct approaches to AI agent/assistant platforms are converging in the market:

1. **AbacusAI ChatLLM** — Unified LLM access + autonomous agents (DeepAgent)
2. **Perplexity** — Real-time search + agentic workflows + enterprise positioning
3. **AntiGravity** — Sovereign agents with persistent identity + local/cloud hybrid + self-hosted option

**Key insight for We Prosper AI:** The competitive advantage lies not in commodity LLM access or search capabilities, but in agents with persistent identity, memory continuity, and privacy control (self-hosted option). This is a structurally different market position than both AbacusAI and Perplexity.

## PART 1: ABACUSAI (ChatLLM Product)

### What is ChatLLM?

**Not just a product.** AbacusAI ChatLLM is the consumer/SMB-facing layer of the AbacusAI platform, which is fundamentally an **end-to-end ML platform** for enterprises. ChatLLM serves as a multi-model orchestration layer + autonomous agent builder.

- **Part of broader platform:** AbacusAI also offers DeepAgent (autonomous agents), CodeLLM (AI pair programmer), AutoML Forecasting, Vision Suite (image/video generation)
- **Core problem solved:** Consolidating access to multiple state-of-the-art LLMs (GPT-5.4, Claude Sonnet 4.6, Gemini 3.1 Pro, Grok 4.1, Llama 4) into a single subscription with integrated tools
- **Target users:** Professionals, students, small teams, and enterprises seeking a unified AI workbench

## PART 2: PERPLEXITY (As an AI Platform)

### What Perplexity Is Beyond Search

Perplexity is **NOT just a search engine**. In 2026, it has evolved into a **multi-modal AI platform with agentic workflows** and is explicitly building an **enterprise product**.

Core evolution:
- Started as search alternative (real-time web results + AI synthesis)
- Evolved to answer engine (with citations, multiple search modes)
- Now: Agent orchestration platform + enterprise software

### Perplexity API & Capabilities

Perplexity launched its **Agent API in March 2026** — a managed runtime for building agentic workflows.

**API Offerings:**

| API | Model | Input | Output | Use Case |
|-----|-------|-------|--------|----------|
| **Sonar (Standard)** | Llama 3.1 70B optimized for search | $1/M tokens | $1/M tokens | Fast web search synthesis |
| **Sonar Deep Research** | Same, optimized for deep research | $2/M input | $8/M output; $2/M citations; $5/1k search queries | In-depth research workflows |
| **Sonar Pro** | Advanced model with broader capabilities | $3/M input | $15/M output | Complex tasks, reasoning |
| **Sonar Reasoning Pro** | Includes reasoning tokens | $2/M input | $8/M output; $3/M reasoning | Multi-step reasoning |
| **Agent API** | Orchestrates multi-model workflows | Custom pricing (not detailed publicly) | N/A | Building autonomous agents with web search, tool execution, reasoning |
| **Search API** | Raw web search results | N/A | $5/1,000 requests | Access to Perplexity's search index |
| **Sandbox API** | Isolated Python/Bash execution | N/A | Custom | Deterministic code execution mid-workflow |

**Key differences from traditional LLM APIs (GPT, Claude):**
- Built-in real-time web search — no need for separate search API
- Agent API handles the full "agentic loop" (retrieval, tool execution, reasoning, fallback)
- Multi-model fallback support (OpenAI, Anthropic, Google, xAI, NVIDIA)
- Citation metadata included natively
- Privacy: Data not logged or used for training

**What makes Perplexity's search different:**

1. **Live crawling** — Owns web crawler, not dependent on Google/Bing APIs
2. **Citations with sources** — Every answer includes verifiable, clickable citations
3. **Search modes:**
   - Quick Search: Fast, simple answers
   - Pro Search: Multi-step reasoning, tailored responses

## PART 3: ANTIGRAVITY (OUR PLATFORM)

### What Makes AntiGravity Different

Unlike AbacusAI (unified LLM access) and Perplexity (real-time search + agents), AntiGravity is built on **three foundational pillars:**

1. **Sovereign agents with persistent identity**
   - Agents have real names (Moriah, Alethea)
   - Personality, voice, opinions
   - Not generic LLM wrappers
   - Identity persists across sessions, relationships, work products

2. **Persistent memory across sessions**
   - Agents remember context, decisions, learnings
   - Memory infrastructure distributed (local + cloud)
   - No context reset between conversations
   - Enables true continuity and relationship building

3. **Local + cloud hybrid + self-hosted option**
   - Can run locally (e.g., Raspberry Pi) for privacy
   - Can sync to cloud for accessibility
   - Not dependent on single cloud provider
   - Self-hosted option as competitive advantage (vs. Perplexity's cloud-only, AbacusAI's cloud-primary)

**Structural market gap:**

The AI agent market is growing at 46% CAGR toward $52.6B by 2030. But the market is fragmenting:

- **Commodity segment:** Unified LLM access (AbacusAI, OpenAI's API, Anthropic's API) — racing to lower prices, features converging
- **Search-first segment:** Real-time search integration (Perplexity, custom build-your-own) — specialized for research and information retrieval
- **Sovereign agent segment:** Agents with identity + memory + privacy (essentially empty) — only early players are AntiGravity

**Market research:** "Sovereign agents" TAM estimates:

- **Enterprise AI agent adoption:** 85% of enterprises deploying some agent by 2025; 40% with task-specific agents by 2026

The unmet need: Agents that are **persistent, identifiable, ownable, and privacy-controlled**.
