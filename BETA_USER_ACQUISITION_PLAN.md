## Feedback Loops

### Daily (Automated)

**Email sequence for new users:**
```
Day 0: Welcome + onboarding help
Day 3: "Did you try uploading? Let me help if stuck"
Day 7: "What's your biggest insight so far?"
Day 14: "Would you recommend to a friend?"
Day 30: "How has Finance Friend changed your thinking?"
```

Purpose: Identify power users (highly engaged) vs. at-risk (silent)

---

### Weekly (Structured)

**Friday Feedback Call:**
- Invite: 3-5 random beta users
- Duration: 30 minutes
- Script: "Tell me what's working and what's broken"
- Record: Common themes
- Action: Fix critical issues by Monday

**Feedback Survey (Typeform):**
```
1. How clear was the onboarding? (1-5)
2. Did the dashboard help you understand your spending? (yes/no)
3. Most useful feature so far: [Coach / Dashboard / Categorization]
4. Biggest friction point: [Open text]
5. Would you recommend? (1-10 NPS)
```

Target: 30% response rate (30 responses/week by week 4)

---

### Monthly (Strategic)

**User Cohort Analysis:**
- Segment: Early signup vs. Late signup
- Analyze: Feature usage, engagement, churn
- Question: What's different about retained vs. churned?

**Roadmap feedback:**
- Share: Finance Friend v3 Phase 2 features
- Ask: Which would you use first?
- Learn: Build for strongest demand signals

## Feedback Prioritization

### Tier 1: Fix Immediately (Blocking usage)
- Cannot login
- Categorization completely broken
- Data not saving
- Coach not responding

**Timeline:** Same day if possible

### Tier 2: Fix This Week
- Confusing onboarding
- Category names unclear
- Mobile layout broken
- Email not sending

**Timeline:** Before next Friday

### Tier 3: Consider for v3
- "Would be cool if..."
- Nice-to-have features
- Cosmetic improvements
- "Like YNAB because..."

**Timeline:** Backlog for Phase 2

### Tier 4: Archive (Nice idea, low demand)
- Requested once, no pattern
- Solved by existing feature
- Out of scope

**Timeline:** Revisit in 6 months

## Risk Mitigation

**Risk: Nobody signs up**
- Mitigation: Warm network guarantees 50-100 users
- Backup: Offer to 10 friends/advisors directly

**Risk: Users don't like it**
- Mitigation: Call them + ask why
- Response: Fast iteration based on feedback
- Backup: Different positioning/feature focus

**Risk: Coach AI doesn't work well**
- Mitigation: Hybrid: Humans answer hard questions week 1
- Backup: Use high-quality templates + human review
- Timeline: Real AI by week 3-4

**Risk: Data security concerns**
- Mitigation: Be transparent about encryption + privacy
- Response: SOC 2 audit plan (Month 3)
- Backup: Offer data export anytime
