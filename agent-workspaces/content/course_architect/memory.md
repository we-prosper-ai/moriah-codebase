## Style Guide

### Core Principles
1. **Begin where students are** — Start with their current knowledge, not your favorite topic
2. **Progress with clear milestones** — Every lesson has a specific learning outcome
3. **Mix theory and practice** — Concept → example → hands-on exercise → real-world application
4. **Measurable outcomes** — Students know what success looks like

## Quality Criteria — How to Know It's Done

| Criterion | How to Measure |
|-----------|----------------|
| Goal is clear | Student knows exactly what they'll achieve |
| Prerequisites clear | No student gets stuck lacking foundations |
| Learning outcomes specific | "Can do X" not "Understands Y" |
| Progression logical | Each lesson builds on previous |
| Mix of teaching methods | Not just videos, includes exercises/projects |
| Time estimates accurate | Match actual student experience |
| Capstone demonstrates mastery | Real work that proves transformation |
| Student reviews positive | Testimonials show success and clarity |

## Pedagogical Principles for Adult Learners

### How Adults Learn Best
1. **Active participation** — Not passive watching
2. **Relevant to their goals** — Why does this matter to them?
3. **Respecting experience** — Build on what they know
4. **Safe failure** — Exercises should be low-stakes
5. **Immediate application** — Try it today, not "in 6 months"

### Module Sequencing Rules
- **Foundational first** — Can't build on quicksand
- **Complexity increases gradually** — Overwhelm kills completion
- **Wins early** — First module should show quick wins
- **Spacing** — Spaced repetition improves retention
- **Coherent narrative** — Modules tell a story, not random topics

### Exercise Design
- **Clear instructions** — No ambiguity
- **Quick wins** — Doable in 15-30 minutes
- **Real scenarios** — Based on actual student work
- **Feedback loop** — They know if they did it right
- **Escalating difficulty** — Each exercise harder than last

### Capstone Project Elements
- **Real world** — Something they'd actually do
- **Significant** — Takes real effort to complete
- **Measurable** — Clear success criteria
- **Feedback** — You or a community member reviews it
- **Portfolio-worthy** — Something they're proud to show

## Learning Sciences References

### Bloom's Taxonomy (Learning Progression)
1. Remember (watch video, read)
2. Understand (explain concept)
3. Apply (try it)
4. Analyze (why did it work?)
5. Evaluate (is this the best approach?)
6. Create (build something new)

*Courses should progress through these levels, not stay at "remember".*

### Cognitive Load Theory
- Don't overwhelm with too much new info at once
- Use visuals to reduce cognitive load
- Spaced practice beats massed practice
- Worked examples before independent practice

### Motivation & Engagement
- **Autonomy:** Some choice in how to learn
- **Competence:** Achievable goals + feedback
- **Relatedness:** Community, connection to peers
- **Intrinsic motivation:** "I want to learn this" not "I have to"
