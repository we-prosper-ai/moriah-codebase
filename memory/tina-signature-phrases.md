## SENTENCE STRUCTURES & PATTERNS

### Opening Patterns
1. **"I have a client who..."** - introduces context
2. **"Please extract..."** - standard instruction opener
3. **"I want you to..."** / **"I need you to..."** - clear directives
4. **"can you help?"** - collaborative tone when uncertain
5. **"okay... [context]"** - transitional, regrouping after feedback

### Correction Patterns
1. **"You left out [X]"** - direct statement of omission
2. **"[Statement]. That's wrong."** - quote + direct contradiction
3. **"take this out [quote]"** - removal instruction with quote
4. **"change [X] to [Y]"** - replacement pattern
5. **"should be: [correct version]"** - prescriptive correction
6. **"It wasn't... it was."** - contrast pattern for corrections
7. **"no. [correction]"** - blunt negation followed by fix

### Explanatory Patterns
1. **"The goal is to [X], then [Y], then [Z]"** - sequential process
2. **"The point of [X] is [Y]"** - clarifies purpose
3. **"We want them to know [X]"** - defines intended message
4. **"For your information, [data]"** - provides missing context
5. **"This is what I got from..."** - introduces external input

### Emphasis Patterns
1. **Ellipses for pause/thinking**: "Like... he said", "Also... is Bouncer a company?"
2. **ALL CAPS for emphasis**: "ALL the possibilities", "WAY TOO MUCH", "ALLLLL", "PER THE CALL", "ONLY"
3. **Bold for key terms**: **while you slept**, **without you becoming a social media circus act**
4. **Parenthetical asides**: "(definitely not the move)", "(if desired)", "(give them an option)"
5. **Repeated emphasis**: "please please please", "ALLLLL"

### Question Patterns (Rhetorical/Strategic)
1. **"What If Your [X]?"** - copywriting hook pattern
2. **"Imagine [scenario]... while you [X]"** - visualization + contrast
3. **"But if you don't understand [X] you can't help"** - conditional challenge
4. **"What am I missing?"** - genuine inquiry
5. **"do you know how to [X]?"** - capability check
